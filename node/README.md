<h1>Node</h1>
Node code to generate fake data to the database, using Faker lib.
<br>
> Must install (via npm):
<br>
> Faker, Faker-br, Mysql 
<br>
<h2>Faker</h2>

```npm install @faker-js/faker --save-dev```
<br>

<h2>Faker br</h2>

```npm install faker-br```
<br>

<h2>Mysql</h2>

```npm install mysql```
